GitHub es una plataforma de desarrollo de código abierto y colaboración que permite a los desarrolladores crear y compartir código. GitHub también ofrece una forma fácil de rastrear y administrar las solicitudes de incorporación de cambios de código (también conocidas como "solicitudes de extracción" o "pull requests") y hacer un seguimiento de los problemas del código (también conocidos como "issues").
